First step:
    Go to : https://openweathermap.org/
    Create an account and get API key

Second step:
    modify the meteo.prop file as 

    "key" : " " 

    to =>

    "key" : "YourNewFreshKey"

Third step:

    Add your city :
    
    Again modify the meteo.prop file as
    
    "city" => "YourCity" 
    
    Like "Paris"
